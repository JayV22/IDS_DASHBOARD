# Hybrid Machine Learning Intrusion Detection System for Cloud Environments

## Overview
This project implements a hybrid machine learning based Intrusion Detection System (IDS) designed for cloud environments. The system detects abnormal network traffic using unsupervised learning algorithms including Autoencoder neural networks and Isolation Forest.

The project uses the CIC-IDS2017 dataset to train models capable of identifying both known and unknown cyber attacks.

## Key Features
• Hybrid anomaly detection system  
• Autoencoder deep learning model  
• Isolation Forest anomaly detection  
• Detection of zero-day attacks  
• Real-time monitoring dashboard using Streamlit  
• Attack severity scoring  
• Scalable architecture suitable for cloud environments  

## Dataset
Dataset Used: **CIC-IDS2017**

Contains realistic network traffic including:
- Benign traffic
- DDoS attacks
- Port scans
- Botnet activity
- Brute force attacks
